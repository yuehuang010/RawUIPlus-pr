# RawUIPlus — Public

Public home for **RawUIPlus** bug reports and third-party license notices.
The application source code is closed and lives in a separate private repository.

## Report a bug

Please [open an issue](../../issues/new/choose) using the **Bug report** template.
Include your app version, clear repro steps, and — if you can — attach the log file
the app writes on each run:

```
%TEMP%\RawUIPlus.YYYYMMDD.PID.log
```
